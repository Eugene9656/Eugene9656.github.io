## 说明
因为虚拟环境使用较多，所以python3.4版本起，官方直接内置了venv包

## 使用
创建虚拟环境
```bash
python -m venv 虚拟环境名（如：zzz）
```
进入目录
```bash
cd 虚拟环境名/Scripts
```
激活虚拟环境
```bash
activate.bat
```
退出虚拟环境
```bash
deactivate.bat
```
